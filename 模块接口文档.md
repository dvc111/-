\# 0. 全局约定



| 项      | 说明                                     |

| ------ | -------------------------------------- |

| 实体 ID  | `entity\\\_id`，字符串，如 `"Q1024"`            |

| 关系 ID  | `relation\\\_id`，字符串，如 `"R017"`           |

| 超关系 ID | `hyper\\\_relation\\\_id`，字符串，如 `"HR03"`     |

| 三元组    | `\\\[head\\\_id, relation\\\_id, tail\\\_id]` 三元数组 |

| 问题 ID  | `question\\\_id`，每次问答唯一标识                 |

| 分数     | `float`，范围 `\\\[0,1]`                     |

| 版本号    | 每个接口 JSON 顶层带 `"schema\\\_version":"0.1"` |

\# 1. Agent + 宏观检索模块 → GNN推理模块



\*\*接口名：`macro\\\_subgraph`\*\*



（由 Agent 实体链接、超关系语义评分以及 Retrieval 宏观子图裁剪共同产出）



```json

{

\&#x20; "schema\\\_version": "0.1",

\&#x20; "question\\\_id": "q\\\_0001",

\&#x20; "question\\\_text": "阿司匹林和布洛芬同时服用会有什么风险？",



\&#x20; "entity\\\_linking": \\\[

\&#x20;   {

\&#x20;     "mention": "阿司匹林",

\&#x20;     "entity\\\_id": "Q1024",

\&#x20;     "confidence": 0.97

\&#x20;   },

\&#x20;   {

\&#x20;     "mention": "布洛芬",

\&#x20;     "entity\\\_id": "Q1088",

\&#x20;     "confidence": 0.95

\&#x20;   }

\&#x20; ],



\&#x20; "topic\\\_entities": \\\[

\&#x20;   "Q1024",

\&#x20;   "Q1088"

\&#x20; ],



\&#x20; "selected\\\_hyper\\\_relations": \\\[

\&#x20;   {

\&#x20;     "relation\\\_id": "HR03",

\&#x20;     "label": "药物相互作用",

\&#x20;     "score": 0.92

\&#x20;   },

\&#x20;   {

\&#x20;     "relation\\\_id": "HR07",

\&#x20;     "label": "禁忌症",

\&#x20;     "score": 0.81

\&#x20;   }

\&#x20; ],



\&#x20; "macro\\\_subgraph": {

\&#x20;   "nodes": \\\[

\&#x20;     {

\&#x20;       "entity\\\_id": "Q1024",

\&#x20;       "label": "阿司匹林",

\&#x20;       "is\\\_topic": true,

\&#x20;       "initial\\\_importance": 0.96

\&#x20;     },

\&#x20;     {

\&#x20;       "entity\\\_id": "Q1088",

\&#x20;       "label": "布洛芬",

\&#x20;       "is\\\_topic": true,

\&#x20;       "initial\\\_importance": 0.94

\&#x20;     },

\&#x20;     {

\&#x20;       "entity\\\_id": "Q2031",

\&#x20;       "label": "胃肠道出血",

\&#x20;       "is\\\_topic": false,

\&#x20;       "initial\\\_importance": 0.72

\&#x20;     }

\&#x20;   ],



\&#x20;   "triples": \\\[

\&#x20;     \\\[

\&#x20;       "Q1024",

\&#x20;       "R017",

\&#x20;       "Q2031"

\&#x20;     ],

\&#x20;     \\\[

\&#x20;       "Q1088",

\&#x20;       "R017",

\&#x20;       "Q2031"

\&#x20;     ]

\&#x20;   ]

\&#x20; },



\&#x20; "max\\\_hops": 3

}

```



\### 字段说明



\* \*\*entity\_linking\*\*：Agent完成实体识别及实体消歧后的结果，作为宏观检索的起始节点。

\* \*\*topic\_entities\*\*：实体链接后的主题实体集合，作为宏观检索和GNN推理的输入。

\* \*\*selected\_hyper\_relations\*\*：Agent依据问题语义评分得到的超关系集合，宏观检索依据高分超关系约束图遍历方向，实现知识领域快速定位。

\* \*\*macro\_subgraph.nodes\*\*：宏观子图中的实体节点。

\* \*\*initial\_importance\*\*：节点初始重要性得分，由实体链接置信度、超关系匹配程度及节点中心性综合计算，为GNN节点重要性分析提供初始化特征。

\* \*\*macro\_subgraph.triples\*\*：宏观子图裁剪后的候选三元组集合，GNN模块直接基于该子图进行图表示学习，无需重新访问完整知识图谱。

\* \*\*max\_hops\*\*：宏观检索过程中允许的最大搜索深度。

\# 2. GNN节点重要性分析 → 候选答案筛选



\*\*接口名：`gnn\\\_reasoning\\\_output`\*\*



（由 GNN 节点重要性计算及候选答案筛选模块产出）



```json

{

\&#x20; "schema\\\_version": "0.1",

\&#x20; "question\\\_id": "q\\\_0001",



\&#x20; "node\\\_importance": \\\[

\&#x20;   {

\&#x20;     "entity\\\_id": "Q2031",

\&#x20;     "importance": 0.91

\&#x20;   },

\&#x20;   {

\&#x20;     "entity\\\_id": "Q1088",

\&#x20;     "importance": 0.74

\&#x20;   },

\&#x20;   {

\&#x20;     "entity\\\_id": "Q1024",

\&#x20;     "importance": 0.68

\&#x20;   }

\&#x20; ],



\&#x20; "candidate\\\_answers": \\\[

\&#x20;   {

\&#x20;     "entity\\\_id": "Q2031",

\&#x20;     "label": "胃肠道出血",

\&#x20;     "prob": 0.76

\&#x20;   },

\&#x20;   {

\&#x20;     "entity\\\_id": "Q2099",

\&#x20;     "label": "肾功能损伤",

\&#x20;     "prob": 0.41

\&#x20;   }

\&#x20; ]

}

```



\### 字段说明



\* \*\*node\_importance\*\*：图神经网络完成消息传播后得到的节点重要性得分，用于衡量各实体与当前问题的相关程度。

\* \*\*candidate\_answers\*\*：依据节点重要性及图结构传播结果，对候选实体进行评分排序，输出 Top-K 候选答案，为后续答案生成提供候选实体集合。

\* \*\*prob\*\*：候选答案最终评分，可用于排序及结果展示。
